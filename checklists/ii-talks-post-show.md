# II Talks — Post-Show Checklist

Use this checklist before the `ii-talks` POST-SHOW mode runs any WooCommerce or Mailchimp actions. All items must be confirmed before Rosy will proceed with live changes.

---

## Event Confirmation
- [ ] Talk title is confirmed
- [ ] Venue is confirmed (UUFVB / SOD / Online)
- [ ] Actual show date is confirmed
- [ ] Approximate number of attendees is noted (for records)

## Participant Tagging
- [ ] Attendee list (names / emails) is available for Mailchimp tagging
- [ ] Participant tag name is confirmed — format: `YYYY-MM-DD [VENUE] IITalks [TalkSlug] Participants`
- [ ] Tag does not already exist in Mailchimp (check to avoid duplicates)

## Coupon Setup
- [ ] A 100%-off coupon code has been created in WooCommerce for the recording product
- [ ] The coupon is scoped to the specific recording product only (not sitewide)
- [ ] Coupon code has been confirmed and is ready to include in the email

## Coupon Email Draft
- [ ] Email subject confirms a free recording is available
- [ ] Coupon code is correctly included in the email body
- [ ] CTA link points to the correct recording product URL
- [ ] Audience segment is set to the participant tag applied in the step above
- [ ] Email is proofread and ready to send

## Audio
- [ ] Vanessa has confirmed the audio file (m4a) is uploaded to Google Drive
- [ ] The Google Drive file name or folder path has been shared with Rosy
- [ ] Audio quality is acceptable (not cutting in/out, levels are good)

## Recording Product
- [ ] Recording product description has been reviewed (update if this is an additional venue for an existing talk)
- [ ] If this is the first recording for this talk title: a new recording product draft is ready in `inbox/`
- [ ] If this talk already has a recording product: WooCommerce product ID is confirmed for the update

## Ticket Cleanup
- [ ] Ticket product is confirmed and its WooCommerce product ID is known
- [ ] Vanessa confirms it is OK to unpublish the ticket product
- [ ] Vanessa confirms it is OK to remove the event from the featured/events page

---

*Clear all items above, then type GREEN LIGHT to proceed with the post-show MCP actions.*
