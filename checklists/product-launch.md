# Product Launch Checklist

Use before **publish**, **upload**, or **go live** on a WooCommerce product.

## Content

- [ ] Title matches your naming convention and past similar products
- [ ] Long description is complete, formatted, and proofread
- [ ] Short description is set (if you use it on the storefront)
- [ ] Categories and tags are correct

## Pricing & inventory

- [ ] Regular price (and sale price if applicable) are correct
- [ ] Stock status / quantity matches reality (or “sold out” is intentional)

## Images

- [ ] Featured image is set and looks good at listing size
- [ ] Gallery images are in the right order and optimized where needed

## Shipping & tax

- [ ] Shipping class matches similar products
- [ ] Tax status is correct for this product type

## SEO & URL

- [ ] Slug (permalink) is clean and matches the product name
- [ ] Meta description or SEO fields are filled if your theme/plugin expects them

## Final checks

- [ ] Product visibility (catalog/search) is what you intend
- [ ] You previewed the product on the live site (or staging) once more

When everything above is done — or you accept the risk — type **GREEN LIGHT** to proceed.
