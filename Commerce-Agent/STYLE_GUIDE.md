## Product Naming Convention
- Format: `[Event Name] — [Product Type]`
  - Example: `Naples Training Event — General Admission`
  - Example: `Vero Beach Intensive — VIP Package`
- Use title case. No all-caps.
- Keep names under 60 characters for clean WooCommerce display.

## Pricing Format
- All prices in USD with two decimal places: `$199.00`
- Sale prices always include a defined end date — no open-ended discounts.

## Shipping Class Defaults
- Training event tickets: `No Shipping` (virtual/digital)
- Physical products (if any): `Standard Shipping`
- Default to matching the most recent similar product's shipping class.

## Tax Status Defaults
- Services and training events: `Taxable` (confirm with Florida DOR rules)
- Digital downloads: `Taxable`
- Default to matching the most recent similar product unless Vanessa specifies otherwise.

## Product Descriptions

### Short Description (1-2 sentences)
- Lead with the benefit, not the feature.
- Include the event location and date if applicable.
- Example: "Join us in Naples for an intensive one-day training. Limited seats available."

### Full Description
- Structure: **What it is** → **Who it's for** → **What's included** → **Logistics**
- Keep paragraphs short (3-4 sentences max).
- Use bullet lists for "What's included" sections.

## Categories
- Training events: `Events > [City Name]`
- Products to be confirmed when first product is added.

## Images
- All images must have descriptive alt-text.
- Preferred format: 1200×628px (landscape) for event banners.
- File naming: `[event-slug]-[image-type].jpg`

## SEO
- SEO title: product name + "| VWPM" — max 60 characters.
- Meta description: 1-2 sentences, include location and primary keyword — max 160 characters.
