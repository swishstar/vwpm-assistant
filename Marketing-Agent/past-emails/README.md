# Past Emails — Reference Library

This folder contains drafts or records of past Mailchimp campaigns that performed well.

The `draft-email` skill reads this folder to match tone, subject line format, structure, and CTA style before drafting any new campaign.

## How to Add a Past Email Record

Save a copy of a successful campaign here as `[campaign-slug].md`.

### Markdown Format (recommended)

```markdown
# [Campaign Name]

**Sent:** YYYY-MM-DD
**Type:** [Newsletter / Promo / Sequence / Announcement]
**Audience:** [Segment]
**Open Rate:** [X%] (optional)

## Subject Line
[Subject line text]

## Preview Text
[Preview text]

## From Name
[Sender name]

## Body
[Full email body]

## Notes
[Any notes on what worked well, tone to match, etc.]
```

The more campaigns you archive here, the more the AI can learn your brand's voice and replicate what works.
